TUTORIAL

1. LEITURA: Para mostrar os itens ocultos que deseja incluir na redação, clique nas reticências (...). Para ocultar, clique no botão ocultar situado após os itens;


2. PRODUÇÃO: Escreva o tema na caixa destinada a ele, selecione um trecho ou conectivo de cada caixa e dois argumentos da caixa de argumentos. Depois, clique na caixa branca abaixo de "Redação". Ao selecionar os trechos, não clique sobre os retângulos tracejados. Para desmarcar, basta outro clique.


3. DADOS / CONTEXTO / PALAVRAS CHAVE /...: Alguns trechos possuem expressões em caixa alta porque precisam ser substituídos por repertórios específicos do tema, os quais o usuário deve saber para preenchimento.


4. MUDANÇA DE ESTRUTURA: O tipo de estrutura usada neste acervo é o mais comum em provas e vestibulares. Para alterá-lo, é necessário criar uma conta e fazer um plano de assinatura no acervo pessoal.
